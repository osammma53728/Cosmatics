# 💄 Cosmetics App (Flutter)

A modern and elegant **Cosmetics Shopping App** built using **Flutter**.  
The app focuses on clean UI, smooth user experience, and a simple checkout flow.

---

## 📱 Features

- Beautiful and modern UI design
- Product listing with images and prices
- Shopping cart functionality
- Checkout screen with:
  - Delivery address
  - Payment method selection
  - Voucher / promo code
  - Payment summary
- Clean and readable Flutter code

---

## 🛠️ Technologies Used

- **Flutter**
- **Dart**
- Material Design
- Custom reusable widgets
